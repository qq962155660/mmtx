# LCN分布式事务解决方案
####  LCN框架。
LCN名称是由早期版本的LCN框架命名，在设计框架之初的框架设计的步骤是如下,各取其首字母得来的LCN命名。
##### *锁定事务单元（lock）
##### *确认事务模块状态(confirm)
##### *通知事务(notify) 
## 	  该项目由LCN发展而来,
##### LCN并不生产事务，LCN只是本地事务的协调工
